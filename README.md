# Lazapee - Online Shopping Platform

Lazapee is a modern online shopping platform that combines the best features of Lazada and Shopee, providing a trusted marketplace exclusively for legitimate sellers. Our platform is designed to offer customers a seamless shopping experience with verified vendors and secure transactions.

## About Lazapee

Lazapee (Lazada + Shopee) is a next-generation e-commerce platform dedicated to connecting legitimate sellers with conscious buyers. We prioritize quality, transparency, and trust by ensuring that only verified and legitimate sellers can operate on our platform.

### Key Features

- **Verified Sellers Only**: All sellers undergo a rigorous verification process to ensure legitimacy and quality
- **Wide Product Range**: Browse diverse product categories from trusted vendors
- **Secure Transactions**: Safe payment processing and buyer protection
- **Fast Checkout**: Simplified and intuitive shopping experience
- **User-Friendly Interface**: Modern and responsive design for all devices
- **Seller Dashboard**: Tools for legitimate sellers to manage their stores and inventory

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/lazapee-app.git
cd lazapee-app
```

2. Install dependencies:
```bash
npm install
```

### Running the Development Server

Start the development server with hot module replacement:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Building for Production

Create an optimized production build:

```bash
npm run build
```

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```

### Linting

Check code quality and consistency:

```bash
npm run lint
```
